How to use these icons

— Paste into Canvas (fastest)

  1. Build your set on the Build a Badge Set tab and click Copy HTML for Canvas.
  2. In your course, create or edit an Item / Document (or an assignment's Description).
  3. In the content editor toolbar, click the source code button — the <> icon
  4. Paste the HTML and save. The badges render inline with your assignment.
The pasted snippet uses only inline styles and inline SVG — no external files, fonts, or JavaScript — so it survives Canvas' HTML sanitizer on both Original and Ultra.
